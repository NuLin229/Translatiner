========================================
    音频翻译软件 - 使用说明
========================================

【软件功能】
将日语/英语/韩语音频自动翻译成中文字幕，
支持网易云音乐风格的滚动字幕显示。

========================================
    第一步：安装必需软件
========================================

需要安装以下3个软件（都是免费的）：

1. Python 3.9（编程语言环境）
   下载地址: https://www.python.org/downloads/release/python-3913/
   ⚠️ 安装时务必勾选 "Add Python to PATH"
   ⚠️ 推荐使用 3.9 版本，兼容性最好

2. Ollama（本地AI翻译引擎）
   下载地址: https://ollama.com/download
   直接下载安装即可

3. FFmpeg（音频处理工具）
   下载地址: https://www.gyan.dev/ffmpeg/builds/
   下载 "ffmpeg-release-essentials.zip"
   
   FFmpeg 详细安装步骤：
   a) 解压下载的zip文件到 D:\ffmpeg（或其他位置）
      解压后应该有 D:\ffmpeg\bin\ffmpeg.exe 这个文件
   
   b) 添加到系统环境变量 PATH：
      - 右键"此电脑" → 属性 → 高级系统设置
      - 点击"环境变量"按钮
      - 在"系统变量"中找到 Path，双击打开
      - 点击"新建"，输入: D:\ffmpeg\bin
      - 点击"确定"保存所有窗口
   
   c) 重启电脑使配置生效
   
   d) 验证安装：打开命令提示符，输入 ffmpeg -version
      如果显示版本信息则安装成功

========================================
    第二步：启动软件
========================================

双击 "启动.bat" 文件

首次启动会自动：
- 安装 Python 依赖库
- 下载 AI 翻译模型（约2GB）
- 下载语音识别模型（约1.5GB）

⚠️ 首次启动需要下载模型，请耐心等待10-30分钟

========================================
    第三步：使用软件
========================================

1. 点击底部的 "添加文件" 按钮
2. 选择音频文件（支持 mp3/wav 格式，最多5个）
3. 等待自动处理（语音识别 + 翻译）
4. 处理完成后自动显示字幕并播放

【操作技巧】
- 点击字幕文字可跳转到对应时间
- 可拖拽底部文件调整播放顺序
- 切换已处理的文件会直接显示，无需重新处理

========================================
    常见问题
========================================

Q: 启动时提示 "Python 未找到"
A: 重新安装 Python 3.9，确保勾选 "Add Python to PATH"

Q: 启动时提示 "Ollama 未找到"
A: 安装 Ollama 后需要重启电脑

Q: 启动时提示 "FFmpeg 未找到"
A: 确保 FFmpeg 的 bin 目录已添加到系统 PATH

Q: 翻译结果是原文（没有翻译）
A: 检查 Ollama 服务是否正在运行
   可以手动运行: ollama serve

Q: 处理很慢
A: 首次处理需要加载模型，后续会快很多
   长音频（>10分钟）处理时间较长，请耐心等待

========================================
    技术支持
========================================

如遇问题，请检查：
1. Python 版本 3.9（推荐）或更高
2. Ollama 服务正在运行
3. FFmpeg 已正确安装
